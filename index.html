Subset Sum Problem
Given a set S = {a₁,…,aₙ} and a target t, does any subset of S sum to exactly t? Use the tabs in the sidebar to switch between DP Mode (fill the table step-by-step), Brute Force (try all 2ⁿ subsets), and Compare (side-by-side).
dp[i][s] = dp[i−1][s] OR dp[i−1][s−aᵢ] (when s ≥ aᵢ)
📖 Algorithm Overview
▶
01
Define the Table
dp[i][s] = TRUE if the first i elements can sum to s. Rows = elements (0..n). Columns = sums (0..t). Start: dp[0][0]=T, all else F.

Why pseudo-polynomial?
DP runs in O(n·t). This looks polynomial — but t can be as large as 2ⁿ. Input size = O(n + log t) bits, so O(n·t) = O(n · 2log t) which is exponential in input bits. That's why it's called pseudo-polynomial and does not prove P = NP.

🔐 NP-Completeness Proof
▶
We prove Subset Sum is NP-complete in two steps: (1) show it is in NP, (2) reduce 3-SAT to it.
Step 1 — Subset Sum ∈ NP
A certificate is a proposed subset T ⊆ S. A polynomial-time verifier checks (a) every element of T exists in S — O(n), and (b) the sum equals t — O(n). Total: O(n). ✓
Verifier(S, t, T):
  assert T ⊆ S      # membership check — O(n)
  assert sum(T) == t  # sum check — O(n)
  return True       # total: O(n) ✓

Forward (φ sat → sum = t): For each xᵢ = true pick vᵢ, else pick v'ᵢ. This gives digit 1 in each variable position. Each clause has ≥1 literal true, contributing ≥1 to its clause digit. Add slacks sⱼ / s'ⱼ to top up each clause digit to exactly 3. Total = t. ✓

Backward (sum = t → φ sat): Digit constraint forces exactly one of {vᵢ, v'ᵢ} per variable → a valid assignment. Each clause digit must reach 3 with ≤ 2 slacks, so ≥ 1 literal row contributes → clause satisfied. ✓

Polynomiality: O(n·m) construction. ∴ 3-SAT ≤ₚ Subset Sum. Combined with Step 1: Subset Sum is NP-complete. ■

Why DP doesn't contradict NP-completeness
DP runs in O(n·t). But t can be as large as 2ⁿ. Input size = O(n+log t) bits, so O(n·t) = O(n·2log t) — exponential in input size. When t is exponentially large, DP is slower than brute force. Pseudo-polynomial ≠ polynomial. NP-completeness stands.
